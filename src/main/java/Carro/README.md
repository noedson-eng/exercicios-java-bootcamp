# 🚗 Simulador de Comportamento de Veículo

Um projeto inicial focado na introdução aos conceitos de **Classes, Atributos e Métodos** em Programação Orientada a Objetos.

## 🛠️ O que foi praticado:
* **Abstração de Objetos:** Criação da classe `Carro` mapeando características físicas (marca, modelo) em atributos e ações reais em métodos.
* **Manipulação de Estados:** Métodos para ligar, desligar, acelerar e frear, controlando a velocidade e o estado atual do veículo em tempo de execução.

## 💻 Como Executar
Rode a classe correspondente ao simulador para interagir com o ciclo de funcionamento do veículo via console.
