# 🐾 Sistema de Controle de Petshop

Este projeto simula o gerenciamento de serviços e rotinas de higiene para animais de estimação, consolidando os pilares de **Abstração** e **Associação de Objetos**.

## 🛠️ Funcionalidades Principais
* **Classe `Pet`:** Representa a entidade do animal, encapsulando informações essenciais como nome, espécie e status de higiene.
* **Classe `MaquinaBanho`:** Responsável por gerenciar o fluxo e o estado dos serviços de banho e tosa, alterando os atributos do objeto `Pet` que interage com ela.
* **Controle de Fluxo:** Implementação de regras para garantir que o serviço seja executado de forma segura e os estados do pet sejam atualizados corretamente.

## 💻 Como Executar
Rode a classe `MainPetShop.java` para acompanhar o ciclo de entrada de um pet no estabelecimento, a execução do serviço e a mudança em seus atributos.
