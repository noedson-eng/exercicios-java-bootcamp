# ☕ Meus Exercícios de Java - Bootcamp Santander

Bem-vindo(a) ao meu repositório de estudos! Aqui estão centralizados todos os desafios de código e exercícios práticos que desenvolvi durante o **Bootcamp Santander Java**.

O objetivo deste repositório é registrar a minha evolução na linguagem Java e consolidar os pilares da **Programação Orientada a Objetos (POO)** utilizando boas práticas de mercado e recursos do Java moderno.

---

## 📂 Organização do Repositório

Para manter o código limpo e escalável, os projetos foram separados por pacotes e contextos específicos. Cada pasta possui sua própria documentação detalhada:

* **[banco](./src/main/java/banco):** Simulador de Conta Bancária com lógica complexa de Cheque Especial, controle de crédito e cobrança automatizada de encargos.
* **[cinema](./src/main/java/cinema):** Sistema de Bilheteria utilizando *Sealed Classes* (Java 21) e polimorfismo dinâmico para cálculo de descontos em ingressos.
* **[sistemadegestao](./src/main/java/sistemadegestao):** Painel de controle de usuários (Gerente, Vendedor, Atendente) aplicando herança e blindagem de níveis de acesso.
* **[petshop](./src/main/java/petshop):** Sistema de gerenciamento de higiene animal focado em associação e manipulação de estados de objetos.
* **[Carro](./src/main/java/Carro):** Exercício inicial focando na introdução a classes, atributos e métodos.

---

## 🛠️ Principais Conceitos Praticados
* **Encapsulamento Estrito:** Proteção de atributos com modificadores `private`, eliminando métodos `set` perigosos em regras de negócio críticas.
* **Herança e Abstração:** Criação de classes mães abstratas para reutilização de código e eliminação de redundâncias (Princípio DRY).
* **Polimorfismo:** Sobrescrita de métodos (`@Override`) para comportamentos dinâmicos em classes filhas.
* **Java Moderno (21+):** Uso de *Sealed Classes* (`sealed`/`permits`) para controle rígido sobre a árvore de herança.
