# Exercícios de Java - Bootcamp 🚀

Repositório criado para salvar meus exercícios de fixação de Orientação a Objetos.

## 🛠️ O que foi desenvolvido:
1. **Conta Bancária:** Sistema simulando uma conta com regras de cheque especial, saques e taxas de 20% automatizadas no depósito.
2. **Simulador de Carro:** Controle de um veículo com validações de segurança por marcha, limite de velocidade inteligente e travas de movimento para curvas.
3. **Controle de Banho de PetShop:** Sistema de gerenciamento de uma máquina de banhos automatizada, controlando níveis de água e shampoo com travas de segurança, além do gerenciamento de entrada, saída e limpeza do status do Pet.

*Foco total em aprender encapsulamento, composição de objetos, lógica condicional avançada e boas práticas de clean code!*
